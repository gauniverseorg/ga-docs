# MinIO

The available properties are:

| Property    | Description         |
| ----------- | ------------------- |
| URL         | URL to MinIO        |
| Port        | Port number         |
| User        | User name           |
| Password    | Password for login  |
| Bucket Name | Bucket of the minIO |
| Is Private  | No anonymous access |
